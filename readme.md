# Rock Paper Scissors (v2)

Welcome to the second version of the classic **Rock Paper Scissors** game! This project is a modern, interactive web-based implementation of the traditional hand game, designed with a clean user interface and smooth user experience.

## 🚀 Live Demo

> https://caelzen.github.io/rock-paper-scissors-v2/

- **Real-time Score Tracking:** Tracks the player's and computer's scores dynamically throughout the session.
- **Visual Feedback:** Clear, instant announcement.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic markup structure for accessibility and clean document layout.
- **CSS3:** Custom styling, layout management (Flexbox/Grid), and interactive hover effects.
- **JavaScript (ES6+):** Application logic, computer move generation, event listeners, and dynamic DOM manipulation.

---

## 🎯 Gameplay Rules

- **Rock** crushes Scissors.
- **Paper** covers Rock.
- **Scissors** cuts Paper.
- The first player to reach **5 points** wins the match! (Adjust if your game logic is different)
